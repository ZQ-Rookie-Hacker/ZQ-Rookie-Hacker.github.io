BloodHoundйЈҹз”Ё

post @ 2026-03-23

1. жүӢеҶҢжҰӮиҝ°

BloodHound жҳҜеҹҹжё—йҖҸжөӢиҜ•дёӯж ёеҝғзҡ„еҸҜи§ҶеҢ–еҲҶжһҗе·Ҙе…·пјҢе…¶йў„жһ„е»әеҲҶжһҗжҹҘиҜўеҸҜзІҫеҮҶе®ҡдҪҚеҹҹеҶ…й«ҳд»·еҖјзӣ®ж ҮгҖҒж”»еҮ»и·Ҝеҫ„гҖҒй«ҳеҚұжқғйҷҗй…ҚзҪ®дёҺеҚҸи®®жјҸжҙһгҖӮжң¬жүӢеҶҢзі»з»ҹжӢҶи§ЈжүҖжңүйў„жһ„е»әжҹҘиҜўзҡ„ж ёеҝғз”ЁйҖ”гҖҒе®һжҲҳд»·еҖјпјҢе№¶з»ҷеҮәеҲҶйҳ¶ж®өдҪҝз”Ёзӯ–з•ҘпјҢеё®еҠ©жё—йҖҸжөӢиҜ•дәәе‘ҳй«ҳж•ҲеҲ©з”ЁжҹҘиҜўз»“жһңи§„еҲ’ж”»еҮ»й“ҫи·ҜгҖӮ

2. йў„жһ„е»әжҹҘиҜўеҲҶзұ»иҜҰи§Ј

2.1 еҹҹеҹәзЎҖдҝЎжҒҜзұ»пјҲDomain Informationпјү

жҹҘиҜўеҗҚз§° ж ёеҝғдҪңз”Ё е®һжҲҳд»·еҖј еӨҮжіЁ
Find all Domain Admins еҲ—еҮәDomain Adminsз»„зҡ„жүҖжңүжҲҗе‘ҳ жҳҺзЎ®еҹҹеҶ…жңҖй«ҳжқғйҷҗиҙҰеҸ·пјҢй”Ғе®ҡжё—йҖҸжңҖз»Ҳзӣ®ж Ү жүҖжңүеҗҺз»ӯжҹҘиҜўзҡ„ж ёеҝғеҸӮз…§зӣ®ж Ү
Map Domain Trusts еҸҜи§ҶеҢ–еҹҹдҝЎд»»е…ізі»пјҲзҲ¶еӯҗеҹҹ/еӨ–йғЁеҹҹ/жһ—дҝЎд»»пјү и§„еҲ’и·Ёеҹҹжё—йҖҸи·Ҝеҫ„пјҢжүҫеҲ°д»ҺеӯҗеҹҹзӘҒз ҙеҲ°зҲ¶еҹҹзҡ„й“ҫи·Ҝ йңҖз»“еҗҲеҹҹжҺ§жӢ“жү‘еҲҶжһҗдҝЎд»»ж–№еҗ‘
Find Computers with Unsupported Operating Systems жҹҘжүҫиҝҗиЎҢе·ІеҒңжӯўж”ҜжҢҒзі»з»ҹпјҲWin7/2008/XPзӯүпјүзҡ„дё»жңә е®ҡдҪҚжңҖжҳ“зӘҒз ҙзҡ„еҲқе§Ӣж”»еҮ»е…ҘеҸЈпјҲжјҸжҙһеӨҡгҖҒиЎҘдёҒе°‘пјү дјҳе…ҲдҪңдёәеҲқе§Ӣжү“зӮ№зҡ„зӣ®ж Үжё…еҚ•

2.2 еҚұйҷ©жқғйҷҗзұ»пјҲDangerous Privilegesпјү

жҹҘиҜўеҗҚз§° ж ёеҝғдҪңз”Ё е®һжҲҳд»·еҖј еӨҮжіЁ
Find Principals with DCSync Rights жҹҘжүҫжӢҘжңүDCSyncжқғйҷҗзҡ„дё»дҪ“пјҲз”ЁжҲ·/з»„/и®Ўз®—жңәпјү жӢҝеҲ°иҜҘжқғйҷҗеҸҜзӣҙжҺҘеҗҢжӯҘжүҖжңүеҹҹиҙҰеҸ·е“ҲеёҢпјҢжҺ§еҲ¶ж•ҙдёӘеҹҹ еҹҹеҶ…жңҖй«ҳеҚұжқғйҷҗпјҢдјҳе…Ҳзә§жңҖй«ҳзҡ„ж”»еҮ»зӣ®ж Ү
Users with Foreign Domain Group Membership жҹҘжүҫеұһдәҺеӨ–йғЁеҹҹз»„зҡ„з”ЁжҲ· дҪңдёәи·Ёеҹҹж”»еҮ»е…ҘеҸЈпјҢеҲ©з”Ёз”ЁжҲ·еңЁе…¶д»–еҹҹзҡ„жқғйҷҗзӘҒз ҙиҫ№з•Ң йңҖз»“еҗҲеҹҹдҝЎд»»е…ізі»еҲҶжһҗжқғйҷҗдј йҖ’
Groups with Foreign Domain Group Membership жҹҘжүҫеҢ…еҗ«еӨ–йғЁеҹҹжҲҗе‘ҳзҡ„з»„ еҲ©з”Ёз»„жқғйҷҗдј йҖ’зү№жҖ§пјҢе®һзҺ°и·ЁеҹҹжЁӘеҗ‘жё—йҖҸ йҮҚзӮ№е…іжіЁй«ҳжқғйҷҗз»„пјҲеҰӮз®ЎзҗҶе‘ҳз»„пјүзҡ„еӨ–йғЁжҲҗе‘ҳ
Find Computers where Domain Users are Local Admin жҹҘжүҫжҷ®йҖҡеҹҹз”ЁжҲ·дёәжң¬ең°з®ЎзҗҶе‘ҳзҡ„дё»жңә жЁӘеҗ‘з§»еҠЁж ёеҝғе…ҘеҸЈпјҢжӢҝеҲ°дё»жңәжқғйҷҗеҗҺзӘғеҸ–й«ҳжқғйҷҗеҮӯиҜҒ дјҳе…ҲеҜ№иҝҷзұ»дё»жңәиҝӣиЎҢжЁӘеҗ‘жү“зӮ№
Find Computers where Domain Users can read LAPS passwords жҹҘжүҫеҸҜиҜ»еҸ–LAPSеҜҶз Ғзҡ„дё»жңә зӣҙжҺҘиҺ·еҸ–дё»жңәжң¬ең°з®ЎзҗҶе‘ҳжқғйҷҗпјҢж— йңҖз ҙи§Ј LAPSдёәжң¬ең°з®ЎзҗҶе‘ҳеҜҶз Ғз®ЎзҗҶе·Ҙе…·пјҢжҳҜй«ҳд»·еҖјзӣ®ж Ү
Find All Paths from Domain Users to High Value Targets з”ҹжҲҗд»Һжҷ®йҖҡеҹҹз”ЁжҲ·еҲ°й«ҳд»·еҖјзӣ®ж Үзҡ„е®Ңж•ҙж”»еҮ»и·Ҝеҫ„ жё—йҖҸж ёеҝғеҜјиҲӘпјҢжҳҺзЎ®вҖңд»Һе“ӘеҸ°жңәгҖҒз”Ёе“ӘдёӘиҙҰеҸ·вҖқжҸҗжқғ еҲқе§Ӣжё—йҖҸйҳ¶ж®өйҰ–йҖүжҹҘиҜў
Find Workstations where Domain Users can RDP жҹҘжүҫжҷ®йҖҡз”ЁжҲ·еҸҜRDPзҡ„е·ҘдҪңз«ҷ зӣҙжҺҘзҷ»еҪ•еҗҺжҸҗжқғгҖҒзӘғеҸ–еҮӯиҜҒ дјҳе…ҲйҖүжӢ©ж ёеҝғдёҡеҠЎдәәе‘ҳзҡ„е·ҘдҪңз«ҷ
Find Servers where Domain Users can RDP жҹҘжүҫжҷ®йҖҡз”ЁжҲ·еҸҜRDPзҡ„жңҚеҠЎеҷЁ жңҚеҠЎеҷЁжүҝиҪҪе…ій”®жңҚеҠЎпјҢжҳ“жҺҘи§Ұй«ҳжқғйҷҗиҙҰеҸ· йҮҚзӮ№е…іжіЁж•°жҚ®еә“гҖҒеә”з”ЁжңҚеҠЎеҷЁ
Find Dangerous Privileges for Domain Users Groups жҹҘжүҫеҹҹз”ЁжҲ·з»„зҡ„еҚұйҷ©зү№жқғпјҲеҰӮSeImpersonatePrivilegeпјү еҲ©з”Ёзү№жқғе®һзҺ°жң¬ең°жҸҗжқғпјҢиҺ·еҸ–зі»з»ҹжқғйҷҗ з»“еҗҲжҸҗжқғе·Ҙе…·пјҲеҰӮJuicyPotatoпјүдҪҝз”Ё
Find Domain Admin Logons to non-Domain Controllers жҹҘжүҫеҹҹз®ЎзҗҶе‘ҳзҷ»еҪ•иҝҮзҡ„йқһеҹҹжҺ§дё»жңә зӘғеҸ–иҝҷдәӣдё»жңәзҡ„еҮӯиҜҒеҸҜзӣҙжҺҘжӢҝеҲ°еҹҹз®Ўжқғйҷҗ жңҖзЁіе®ҡзҡ„жҸҗжқғж–№ејҸд№ӢдёҖпјҢдјҳе…ҲеҲҶжһҗ

2.3 Kerberos еҚҸи®®зұ»пјҲKerberos Interactionпјү

жҹҘиҜўеҗҚз§° ж ёеҝғдҪңз”Ё е®һжҲҳд»·еҖј еӨҮжіЁ
Find Kerberoastable Members of High Value Groups жҹҘжүҫй«ҳд»·еҖјз»„дёӯеҸҜиў«Kerberoastingж”»еҮ»зҡ„жҲҗе‘ҳ дјҳе…Ҳж”»еҮ»й«ҳжқғйҷҗжңҚеҠЎиҙҰеҸ·пјҢжҸҗеҚҮжҸҗжқғж•ҲзҺҮ й«ҳд»·еҖјз»„еҢ…жӢ¬еҹҹз®Ўз»„гҖҒдјҒдёҡз®ЎзҗҶе‘ҳз»„зӯү
List all Kerberoastable Accounts еҲ—еҮәжүҖжңүеёҰSPNзҡ„з”ЁжҲ·/жңҚеҠЎиҙҰеҸ· еҸ‘иө·Kerberoastingж”»еҮ»пјҢзҰ»зәҝз ҙи§ЈNTLMе“ҲеёҢ йңҖз»“еҗҲImpacketзӯүе·Ҙе…·иҜ·жұӮжңҚеҠЎзҘЁжҚ®
Find Kerberoastable Users with most privileges жҹҘжүҫжқғйҷҗжңҖй«ҳзҡ„еҸҜKerberoastingз”ЁжҲ· зІҫеҮҶе®ҡдҪҚй«ҳд»·еҖјж”»еҮ»зӣ®ж ҮпјҢеҮҸе°‘ж— ж•Ҳз ҙи§Ј з»“еҗҲжқғйҷҗеӣҫи°ұеҲҶжһҗз”ЁжҲ·жүҖеұһз»„
Find AS-REP Roastable Users (DontReqPreAuth) жҹҘжүҫж— йңҖKerberosйў„и®ӨиҜҒзҡ„з”ЁжҲ· еҸ‘иө·AS-REP Roastingж”»еҮ»пјҢзӣҙжҺҘиҺ·еҸ–NTе“ҲеёҢ ж”»еҮ»й—Ёж§ӣжһҒдҪҺпјҢж— еүҚзҪ®жқғйҷҗд№ҹеҸҜжү§иЎҢ
