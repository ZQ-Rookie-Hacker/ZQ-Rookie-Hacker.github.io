基础命令速查

命令 功能
git init 初始化仓库
git add . 添加所有修改到暂存区
git commit -m "msg" 提交到本地仓库
git status 查看状态
git log --oneline 简洁查看提交历史
git branch 查看本地分支
git branch -a 查看所有分支（含远程）
git checkout -b <branch> 创建并切换分支
git branch -d <branch> 删除已合并分支
git branch -D <branch> 强制删除分支
